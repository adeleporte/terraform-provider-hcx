This TF file automates the configuration of a HCX setup. It manages creation/update/deletion of :
* HCX activation and configuration at VMC side
* Site pairing
* Network profiles (Management, vMotion and Uplink)
* Compute profile
* Service Mesh
* L2 Extension

## Usage example

  required_providers {
    hcx = {
      source = "adeleporte/hcx"
    }
  }
}
